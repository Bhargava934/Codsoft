# Codsoft
This repository contains beginner-friendly Python projects developed as part of my learning journey and internship tasks. These projects helped me improve my understanding of Python programming concepts, logic building, and problem-solving skills.

📌 Projects Included
1. 📝 To-Do List Application

A simple command-line To-Do List application that allows users to:

Add tasks
View tasks
Remove completed tasks

Concepts Used: Lists, Functions, Loops, Conditional Statements

2. 🧮 Calculator

A basic calculator program that performs arithmetic operations such as:

Addition
Subtraction
Multiplication
Division

Concepts Used: Functions, User Input, Conditional Statements

3. 🔐 Password Generator

A Python program that generates strong and random passwords using:

Letters
Numbers
Special Characters

Concepts Used: Random Module, Strings, Loops

4. ✊✋✌ Rock Paper Scissors Game

A fun command-line game where the user plays against the computer.

Features:

Random computer choices
Score checking
Winner announcement

Concepts Used: Random Module, Conditional Logic, Loops

🛠 Technologies Used
Python 3
VS Code / PyCharm
